# Installation

Complete guide to installing and setting up Leaf.

## Prerequisites

- **Node.js** version 18 or higher
- **npm**, **bun**, **pnpm**, or **yarn** package manager

## Quick Install (Recommended)

For most users, install the CLI which includes everything:

:::code-group

```bash [npm]
npm install -D @sylphx/leaf-cli
```

```bash [bun]
bun add -D @sylphx/leaf-cli
```

```bash [pnpm]
pnpm add -D @sylphx/leaf-cli
```

```bash [yarn]
yarn add -D @sylphx/leaf-cli
```

:::

**What's included:**
- `@sylphx/leaf` - Core framework
- `@sylphx/leaf-theme-default` - Default theme
- CLI commands: `leaf dev`, `leaf build`, `leaf preview`

## Manual Setup (Advanced)

For advanced use cases like custom build pipelines or monorepos.

### 1. Install Core Packages

```bash
npm install @sylphx/leaf @sylphx/leaf-theme-default
```

### 2. Create Project Structure

```
my-docs/
├── docs/              # Your markdown files
│   └── index.mdx
├── src/
│   └── main.tsx       # React entry point
├── leaf.config.ts     # Optional configuration
├── vite.config.ts     # Vite configuration
└── package.json
```

### 3. Configure Vite

Create `vite.config.ts`:

```typescript
import { defineConfig } from "vite";
import { createLeafPlugin, routesPlugin, loadConfig } from "@sylphx/leaf";
import { resolve } from "node:path";

const docsDir = resolve(process.cwd(), "docs");
const config = await loadConfig(process.cwd());

export default defineConfig({
  plugins: [routesPlugin(docsDir), ...createLeafPlugin(config)]
});
```

### 4. Create Entry Point

Create `src/main.tsx`:

```typescript
import { StrictMode } from "react";
import { createRoot } from "react-dom/client";
import { BrowserRouter, Route, Routes } from "react-router-dom";
import { Layout } from "@sylphx/leaf-theme-default";
import "@sylphx/leaf-theme-default/style.css";

// @ts-expect-error - Generated by Vite plugin
import { routes } from "virtual:leaf/routes";
import config from "./config.json";

createRoot(document.getElementById("root")!).render(
  <StrictMode>
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Layout config={config} />}>
          {routes.map((route: any) => (
            <Route key={route.path} path={route.path} element={<route.component />} />
          ))}
        </Route>
      </Routes>
    </BrowserRouter>
  </StrictMode>
);
```

### 5. Add Scripts

Add to `package.json`:

```json
{
  "scripts": {
    "dev": "vite",
    "build": "vite build",
    "preview": "vite preview"
  }
}
```

### 6. Create Your First Page

Create `docs/index.mdx`:

```markdown
---
title: Welcome
---

# Welcome to My Docs

This is my documentation site built with Leaf!
```

For complete examples, see the [examples](https://github.com/sylphxltd/leaf/tree/main/examples) directory.

## Package Comparison

Understanding when to use each package:

| Package | Use Case | Installation |
|---------|----------|--------------|
| **@sylphx/leaf-cli** | ✅ Most users | `npm install -D @sylphx/leaf-cli` |
| **@sylphx/leaf** + **@sylphx/leaf-theme-default** | Custom builds, monorepos | `npm install @sylphx/leaf @sylphx/leaf-theme-default` |

**Recommendation:** Start with `@sylphx/leaf-cli`. Only use manual setup for advanced customization.

## Troubleshooting

### "Cannot find module '@sylphx/leaf-cli'"

Make sure you installed the CLI package:

```bash
npm install -D @sylphx/leaf-cli
```

### "leaf: command not found"

Use `npx` to run the CLI:

```bash
npx leaf dev
```

Or add `node_modules/.bin` to your PATH.

### TypeScript Errors

Install type definitions:

```bash
npm install -D @types/node
```
