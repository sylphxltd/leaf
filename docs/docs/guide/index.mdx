## Getting Started

New to Leaf? Start here:

Learn how to install Leaf and set up your first documentation site. Covers:

### [Configuration](/guide/configuration)

Understand Leaf's configuration system:
- Config file structure (`leaf.config.ts`)
- Site metadata (title, description)

Master Leaf's powerful features:

- Tables and more

Customize Leaf's appearance:

- Creating custom components
- Building a custom theme from scratch
- Theme configuration options
