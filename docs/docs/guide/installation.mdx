# Installation

Leaf is available as an npm package and can be installed in any React project.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** version 18 or higher
- **Bun** or **npm/pnpm/yarn** package manager

## Quick Install

:::code-group

```bash [npm]
npm install @sylphx/leaf @sylphx/leaf-theme-default
```

```bash [bun]
bun add @sylphx/leaf @sylphx/leaf-theme-default
```

```bash [pnpm]
pnpm add @sylphx/leaf @sylphx/leaf-theme-default
```

```bash [yarn]
yarn add @sylphx/leaf @sylphx/leaf-theme-default
```

:::

## Create a New Project

If you're starting from scratch, you can create a new Leaf project using our starter template:

:::code-group

```bash [npm]
npm create leaf@latest my-docs
cd my-docs
npm install
npm run dev
```

```bash [bun]
bun create leaf my-docs
cd my-docs
bun install
bun run dev
```

:::

## Manual Setup

If you prefer to set up Leaf manually:

### 1. Create Project Structure

```
my-docs/
├── docs/              # Your markdown files
│   ├── index.mdx
│   ├── guide/
│   └── api/
├── src/
│   └── main.tsx       # React entry point
├── index.html         # HTML template
├── leaf.config.ts
├── vite.config.ts
└── package.json
```

### 2. Configure Leaf

Create `leaf.config.ts`:

```typescript
import { defineConfig } from "@sylphx/leaf";

export default defineConfig({
  title: "My Documentation",
  description: "My awesome documentation site",
  theme: {
    nav: [
      { text: "Guide", link: "/guide" },
      { text: "API", link: "/api" }
    ],
    sidebar: [
      {
        text: "Getting Started",
        items: [
          { text: "Introduction", link: "/" },
          { text: "Installation", link: "/installation" }
        ]
      }
    ]
  }
});
```

### 3. Create Entry Point

Create `src/main.tsx`:

```typescript
import { StrictMode } from "react";
import { createRoot } from "react-dom/client";
import { BrowserRouter, Route, Routes } from "react-router-dom";
import { Layout } from "@sylphx/leaf-theme-default";
import "@sylphx/leaf-theme-default/style.css";

// @ts-expect-error - Generated by Vite plugin
import { routes } from "virtual:leaf/routes";
import config from "./config.json";

createRoot(document.getElementById("root")!).render(
  <StrictMode>
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<Layout config={config} />}>
          {routes.map((route: any) => (
            <Route key={route.path} path={route.path} element={<route.component />} />
          ))}
        </Route>
      </Routes>
    </BrowserRouter>
  </StrictMode>
);
```

### 4. Configure Vite

Create `vite.config.ts`:

```typescript
import { defineConfig } from "vite";
import { createLeafPlugin, routesPlugin, loadConfig } from "@sylphx/leaf";
import { resolve } from "node:path";

const docsDir = resolve(process.cwd(), "docs");
const config = await loadConfig(process.cwd());

export default defineConfig({
  plugins: [routesPlugin(docsDir), ...createLeafPlugin(config)]
});
```

### 5. Add Scripts

Add to `package.json`:

```json
{
  "scripts": {
    "dev": "vite",
    "build": "bun run build.ts",
    "preview": "vite preview"
  }
}
```

### 6. Create Your First Page

Create `docs/index.mdx`:

```markdown
---
title: Welcome
---

# Welcome to My Docs

This is my documentation site built with Leaf!
```

## Verify Installation

Start the development server:

:::code-group

```bash [npm]
npm run dev
```

```bash [bun]
bun run dev
```

:::

Open your browser to `http://localhost:5173` to see your site!
