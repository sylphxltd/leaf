## Templates

### Starter Template

Clone the official starter template:

```bash
bunx degit sylphxltd/leaf/examples/starter my-docs
cd my-docs
bun install
bun run dev
```

### Blog Template

Documentation site with blog features:

```bash
bunx degit sylphxltd/leaf/examples/blog my-blog
cd my-blog
bun install
bun run dev
```
