# What is Leaf? <Badge type="tip" text="NEW" />

Leaf is a **modern, fast, and feature-rich documentation framework** built with React that achieves **100% feature parity with VitePress**.

## Features at a Glance

::: tip Lightning Fast
Built on **Vite** and **Bun** for blazing fast builds (~400ms) and hot module replacement.
:::

::: warning Production Ready
All core features implemented and tested. Ready for production use!
:::

## Why Leaf?

1. **React Ecosystem** - Full access to React components and libraries
2. **Modern Tooling** - Bun runtime for faster installs and execution
3. **Lightweight** - Minimal dependencies with Sylphx tools (Zen + Silk)
4. **Type Safe** - 100% TypeScript throughout
5. **VitePress Parity** - All features you love from VitePress
