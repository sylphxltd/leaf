# Installation Guide

Install ReactPress with your favorite package manager:

## Using Bun

```bash
bun add @sylphx/reactpress
```

## Using npm

```bash
npm install @sylphx/reactpress
```

## Using pnpm

```bash
pnpm add @sylphx/reactpress
```

## Next Steps

After installation, check out the [Getting Started](/getting-started) guide.
